## Usage

```js
var pkg = require('load-pkg')();
console.log(pkg.name);
//=> 'load-pkg'
```

Optionally specify a cwd or filepath to start from:

```js
var pkg = require('load-pkg')('a/b/c');
console.log(pkg.name);
//=> 'foo'
```
